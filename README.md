# Tara Kumar Gajula — Portfolio Site

A high-impact, dark-themed portfolio built for GitHub Pages.

## 🚀 Deploy to GitHub Pages in 3 steps

### 1. Create a new repository
Name it exactly: `gajulatarak.github.io`

### 2. Upload the files
- Copy `index.html` into the repo root

### 3. Enable GitHub Pages
- Go to repo **Settings → Pages**
- Source: **Deploy from a branch**
- Branch: `main` → `/ (root)`
- Click **Save**

Your site will be live at: **https://gajulatarak.github.io**

## ✨ Features
- Animated hero with terminal-style JSON display  
- Interactive radar + bar charts (Chart.js)
- Proficiency bars with scroll-triggered animation
- Career timeline visualization
- Impact metrics strip
- Project cards with live GitHub links
- Certifications & awards grid
- Fully responsive (mobile + desktop)
